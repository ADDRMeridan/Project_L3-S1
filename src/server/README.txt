######  Faire fonctionner le serveur  ######
-Télécharger netBeans
-Créer un nouveau projet (File->New Project->Java->Java Application)
-Ajouter au <default package>(ou a n'importe quelle package que vous avez créer dans le Source Packages) tout les fichiers .java du repertoire Serveur
-Faire un clique droit sur Source Packages puis cliquer sur Properties aller dans Run puis dans Main class selectionner Server
-Faire un clique droit sur Libraries ensuite cliquer sur Add JAR/Folder et selectionner mysql-connector-java-5.1.44-bin.jar
-Pour lancer le serveur il vous suffit maintenant de cliquer sur Run Project (la fleche jaune dans la barre d'outil)
